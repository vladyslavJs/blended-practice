Методи масивів :

arrow.push - додає нове значення в кінець масиву та повертає нову довжину масиву

Синтаксис : arr.push(element1, ..., elementN)

Приклад :

var sports = ["футбол", "бейсбол"];

var total = sports.push("американський футбол", "плавання");

console.log(sports); // ['футбол', 'бейсбол', 'американський футбол', 'плавання']

console.log(total); // 4
